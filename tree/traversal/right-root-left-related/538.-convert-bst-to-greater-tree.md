_**My Solutions:**_

方法1：recursive

```text
class Solution {
    
    int sum = 0;
    
    public TreeNode convertBST(TreeNode root) {
        
        if (root != null) {
            convertBST(root.right);
            sum += root.val;
            root.val = sum;
            convertBST(root.left);
        }
        return root;      
    }
}
```

方法2：iterative stack

```text
//iterative
    public TreeNode convertBST(TreeNode root) {
        
        Stack<TreeNode> s = new Stack<>();
        TreeNode node = root;
        int sum = 0;
        while (!s.isEmpty() || node != null) {
            while (node != null) {
                s.add(node);
                node = node.right;
            }
            node = s.pop();
            sum += node.val;
            node.val = sum;
            node = node.left;
        }
        return root;
    }
```
